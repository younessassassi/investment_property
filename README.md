# Rental Property Analyzer

Early prototype. Features:

- Input form for property + financing assumptions
- Yearly projection cash vs financed
- IRR estimate & cumulative cash flow charts

## Dev Setup

Install deps then run dev server.

```
npm install
npm run dev
```

Open http://localhost:5173
